# MemoNext — Product Roadmap

MemoNext is a personal learning platform that combines flashcards, AI generation, and learning science techniques to help users study efficiently.

The project evolves through several phases, gradually transforming a simple flashcard tool into an adaptive AI learning system.

---

# Phase 1 — Core Flashcard System (MVP)

**Goal:** Allow users to create decks, add cards, and study them efficiently.

**Features:**

**Users**
• Multiple users
• Admin user
• Local user selection
• Future authentication support

**Decks**
• Create deck
• Edit deck metadata
• Multiple decks per user
• Deck descriptions

**Flashcards**
• Manual flashcard creation
• Short answers
• Detailed explanations
• Difficulty tags

**Study Mode**
• Tap to reveal answers
• Swipe navigation
• Keyboard shortcuts
• Mobile-friendly interface
• Fullscreen study layout

---

# Phase 2 — AI Flashcard Generation

**Goal:** Accelerate learning by generating flashcards automatically.

**Generation methods:**

**Topic generation**
• Generate cards from a topic
• Example: "Greek mythology"

**Webpage generation**
• Paste a URL (e.g., Wikipedia)
• Extract content and generate flashcards

**Text import**
• Paste lecture notes or articles
• Automatically convert to flashcards

**Adjustable generation settings**
• Number of cards
• Answer detail level
• Difficulty level

---

# Phase 3 — Study Engine (Learning Science)

**Goal:** Improve long-term retention using proven learning techniques.

**Spaced repetition**
• Card difficulty rating (Again / Hard / Good / Easy)
• Review scheduling
• Next review timestamps
• Review history

**Daily learning queue**
• "Cards due today"
• Study streaks

**Active recall**
• Prompt-based recall using flashcards
• Answer reveal interaction

---

# Phase 4 — AI Tutor

**Goal:** Provide contextual explanations for difficult concepts.

**AI tutoring features:**

**Explain card**
• Provide deeper explanation of a concept

**Examples**
• Generate real-world examples

**Mnemonics**
• Suggest memory aids

**Follow-up questions**
• Allow interactive learning

---

# Phase 5 — Learning Analytics

**Goal:** Give users insight into their learning progress.

**Metrics:**

**Learning progress**
• Cards mastered
• Retention rate
• Difficulty distribution

**Study activity**
• Time spent studying
• Daily streaks

**Performance insights**
• Weak topics
• Review accuracy

---

# Phase 6 — Personal Knowledge System

**Goal:** Allow users to organize knowledge more flexibly.

**Features:**

**Tags**
• Organize flashcards by topic

**Search**
• Search across decks and cards

**Card linking**
• Connect related concepts

---

# Phase 7 — Monetization

**Goal:** Sustain development of the platform.

**Possible pricing model:**

**Free**
• Limited decks
• Limited AI generation

**Pro**
• Unlimited decks
• Unlimited AI generation
• Learning analytics
• AI tutoring

---

# Phase 8 — AI Learning Experiments

**Goal:** Use the platform to study learning effectiveness.

**Possible experiments:**

• Does explanation improve retention?
• Do longer answers reduce recall accuracy?
• Does spaced repetition interval tuning improve memory?

This allows the platform to evolve into an AI-assisted learning laboratory.

---

# Long-Term Vision

MemoNext aims to become an adaptive learning platform combining:

• Flashcards  
• AI-generated learning materials  
• Learning science techniques  
• Personalized study analytics  

The long-term goal is to create an intelligent study system that adapts to each learner's memory and knowledge gaps.
