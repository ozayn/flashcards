# Product Strategy

## Product Vision

The platform transforms knowledge into structured learning material using AI.

Instead of manually creating flashcards, users can convert:

- notes
- articles
- textbooks
- lectures
- websites

into organized flashcard decks.

---

## Core Idea

**Knowledge → AI → Structured Learning**

**Example:**

```
User pastes a Wikipedia page
↓
AI extracts key concepts
↓
AI generates flashcards
↓
User studies with spaced repetition
```

---

## How This Platform Is Different

**Traditional flashcard apps:**

- require manual card creation
- focus primarily on spaced repetition

**This platform focuses on:**

- AI-generated flashcards
- automatic concept extraction
- flashcards from many sources
- personalized learning
- model experimentation

---

## Competitive Landscape

**Existing tools:**

- Anki
- Quizlet
- RemNote
- Mem.ai
- Notion AI

**Those tools focus on:**

- manual flashcards
- note-taking
- knowledge graphs

**This platform focuses on:**

AI knowledge conversion—turning any source into structured learning material automatically.

---

## Long-Term Vision

Turn the platform into an **AI learning laboratory**.

**Goals:**

- experiment with different LLMs
- evaluate explanation quality
- measure learning efficiency
- optimize flashcard generation

**The LLM router architecture enables:**

- model comparison
- fallback providers
- cost-aware routing
- local model experimentation

---

## Future Differentiators

**Potential features:**

- AI Course Builder
- Concept graph flashcards
- Personalized decks based on user profile
- Flashcards generated from any knowledge source
